Detailed Description

Glavna datoteka za projekt senzorskog sustava s servo motorom i LCD-om.

Sustav koristi tri ultrazvučna senzora za detekciju prepreka, upravlja servo motorom, LED indikatorima i LCD zaslonom. Omogućuje prijelaz u sleep mode radi uštede energije.

◆ measureDistance()

long measureDistance ( int  trigPin,
int  echoPin 
)

Mjeri udaljenost pomoću HC-SR04 ultrazvučnog senzora.

Parameters
trigPinPin za trigger signal (OUTPUT)
echoPinPin za echo signal (INPUT)
Returns
Udaljenost u centimetrima

Funkcija šalje 10μs impuls na trigger pin, zatim mjeri širinu pulseIn() signala na echo pinu i pretvara u centimetre

Note
Brzina zvuka je 343 m/s (0.034 cm/μs), dijelimo s 2 jer je signal putovao do objekta i natrag
